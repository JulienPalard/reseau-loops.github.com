Le réseau LoOPS a le plaisir de vous proposer une demi-journée sur le thème de l'intégration continue.

Cette demi-journée se tiendra le mardi 23 janvier 2018 de 13h30 à 17h à l'INRIA de Saclay - Bâtiment Turing - Salle Henri Poincarré.

Après une introduction aux tests logiciels et à l'intégration continue, nous allons mettre en place sur un exemple simple archivé sur un gitlab.inria.fr (écrit en C++, Java ou Python au choix), à la fois localement, et sur une instance de Jenkins via l'infrastructure d'intégration continue de l'INRIA (ci.inria.fr).

Les programmeurs de tous horizons sont bienvenus (ingénieurs, doctorants, chercheurs...).

La présentation et le support pour les travaux pratiques sont en Anglais, mais seront exposés en Français.

Pré-requis

  • la pratique d'au moins un langage de programmation (C++, Java ou Python)
  • connaissance de l'outil de versionnement git
  • chaque personne devra au préalable suivre cette procédure pour pouvoir participer aux travaux pratiques.

Inscriptions

L'inscription est gratuite mais obligatoire pour nous permettre de préparer au mieux cette demi-journée. Le nombre de places est limitée à 15 personnes. Veillez également à obtenir un ordre de mission (éventuellement sans frais) auprès de votre unité ou de votre tutelle.

Merci de vous inscrire via ce formulaire.

Programme

  • 13h30-14h00 Introduction à l'Intégration Continue et présentation de Jenkins
  • 15h00-18h00 Mise en pratique à partir d'un code fourni, au choix: C++, Java ou Python
    • Prise en main du code fourni, compilation en local et sur la plateforme d'intégration continue
    • Implémentation de tests unitaires localement et sur la plateforme d'intégration continue, mise en place d'un tableau de bord de résultats
    • Mise en œuvre de techniques d'analyse statique et de couverture des tests
