Le réseau LoOPS a le plaisir de vous proposer une demi-journée sur le thème de l'intégration continue.

Cette demi-journée se tiendra le mardi 15 janvier 2018 de 13h30 à 18h à l'INRIA de Saclay - Bâtiment Turing - Salle Henri Poincaré.

Après une introduction aux tests logiciels et à l'intégration continue, nous allons mettre en place sur un exemple simple archivé sur l'instance gitlab gitlab.inria.fr (écrit en C++, Java ou Python au choix), à la fois localement, puis via des GitLab runners opérés par GitLab CI/CD.

Les programmeurs de tous horizons sont bienvenus (ingénieurs, doctorants, chercheurs...).

La présentation et le support pour les travaux pratiques sont en Anglais, mais seront exposés en Français.

Pré-requis

  • la pratique d'au moins un langage de programmation (C++, Java ou Python)
  • connaissance de l'outil de versionnement git
  • chaque personne devra au préalable suivre cette procédure pour pouvoir participer aux travaux pratiques.

Inscriptions

L'inscription est gratuite mais obligatoire pour nous permettre de préparer au mieux cette demi-journée. Le nombre de places est limitée à 15 personnes. Veillez également à obtenir un ordre de mission (éventuellement sans frais) auprès de votre unité ou de votre tutelle.

Désolé, l'inscription à cette journée est déjà remplie. Si vous êtes intéressés pour que l'on rejoue cette demi journé, merci de nous envoyer un mail à loops-cp AT groupes.renater.fr si vous êtes intéressés pour que l'on rejoue cette session.

Programme

  • 13h30-14h00 Introduction à l'Intégration Continue et présentation de GitLab CI / CD
  • 15h00-18h00 Mise en pratique à partir d'un code fourni, au choix: C++, Java ou Python
    • Prise en main du code fourni, compilation en local, puis via l'intégration continue
    • Implémentation de tests unitaires localement, puis via l'intégration continue, mise en place d'un tableau de bord de résultats
    • Mise en œuvre de techniques d'analyse statique et de couverture des tests
    • Utilisation de conteneurs docker pour l'intégration continue
