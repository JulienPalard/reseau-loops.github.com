Café LoOPS du 5 septembre 2017

Attention : RDV exceptionnellement à l'IAS

Chaque premier mardi du mois, nous nous retrouvons au PROTO204, de 13h à 14h, pour discuter de développement logiciel. Pour ce café de rentrée, nous aurons le plaisir d'accueillir Alexis Jeandet qui nous présentera un outil prometteur, l'outil Meson. Meson possède de nombreuses fonctionnalités intéressantes:

  • Wrapdb qui permet d'automatiquement télécharger le code source d'une dépendance si elle n'est pas installée.
  • La gestion automatique des tests unitaires avec la couverture de code automatique.
  • Le support de GTK et Qt4/5
  • Cross compilation ultra simple
  • Generation de header de config ultra simple

  • The Meson Build system
